# Changelog

Full, signed release notes are published on the
[GitHub Releases](https://github.com/liatrio/autogov/releases) page.

## v1.0.0 — first stable release

Marks the first stable release of the `autogov` CLI. From v1.0.0 onward this
project follows [Semantic Versioning](https://semver.org): breaking changes to
the CLI's commands, flags, and output contract bump the major version.
